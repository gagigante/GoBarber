## :runner: How to run

> ### Requirements

- Node
- NPM or Yarn

Install dependencies
```
  $ yarn

  # or

  $npm install

```

After that, create a file `.env` based in `.env.example`
```
REACT_APP_API_URL=http://localhost:3333
```

Finally, Start the server
```
  $ yarn start

  # or

  $ npm start
```
